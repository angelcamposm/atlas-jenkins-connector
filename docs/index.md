## Features

- **Resource-Based:** Cleanly separated resources for Jobs, Builds, and Users.
- **Type-Safe:** Built with PHP 8.5 features and Level 9 PHPStan analysis.
- **Recursive Discovery:** Find all jobs across nested folders.
- **Subfolder Support:** Automatic Jenkins path resolution (`job/folder/job/job`).
- **100% Tested:** Comprehensive unit tests for all features.
