# Contributing

Thank you for considering contributing to the Atlas Jenkins Connector!

## Development Process

1. Fork the repository and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes (`composer test`).
5. Ensure PHPStan analysis passes (`composer analyse`).
6. Format your code (`composer format`).
7. Issue that pull request!

## Coding Standards

This project follows the PER Coding Style (PER-CS). You can use the following command to format your code:

```bash
composer format
```

## Testing

You can run the test suite using:

```bash
composer test
```

## Static Analysis

You can run PHPStan using:

```bash
composer analyse
```
